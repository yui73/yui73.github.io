# 二级指针

**笔记**：

 1. **p 指向值
 2. *p 指向1级指针地址
 3. p 二级指针地址

**代码测试**：

```cpp
#include<cstdio>
#include<iostream>
#include<cstring>
#include<cstdlib>
using namespace std;

void Swap(int **i,int **j)
{
    printf("Before:%d %d\n",*i,*j);
    int *t;
    t=*i;
    *i=*j;
    *j=t;
    printf("After:%d %d\n",*i,*j);
}

int main()
{
    int a=4,b=5;
    int *p1=&a,*p2=&b;
    int **p3=&p1,**p4=&p2;
    printf("BeforeSwap:%d %d\n",p1,p2);
    Swap(p3,p4);
    printf("AfterSwap:%d %d\n",p1,p2);
    printf("%d %d",*p1,*p2);
    return 0;
}
```
