1 给字体换颜色
```
<font size='3' color='#191970'></font>
```
2 插入希腊字母
例：&lambda;
```
$\lambda$ 
//该方法无法转义成HTML，但仅Markdown是支持的
//因此推荐直接使用HTML支持的字符如下:
&lambda;
```
![Markdown字符]( /imgInPosts/MarkdownUsage/Markdown.png )
3 插入上下标
H<sub>2</sub>O
```
H<sub>2</sub>O
```
H<sup>2</sup>
```
H<sup>2</sup>
```
